    Instructions

    To import Info Select items....

    1. if you have already imported IS files, you need to CLEAR THE EXISTING IS IDS THAT WERE IMPORTED THE FIRST TIME. That's because they are not unique across imports. There is not a built-in mechanism to do this yet.
    2. export a chunk of an IS file to HTML
    3. move that to the same directory with parseIShtml.py. The main selector file should be in that directory, and the supporting files in a subdirectory of that directory.
    4. run parseIShtml.py over the tree of web pages
    5. use the resulting output file to enter here
    6. before you run the import, be sure to create the Project in Benk that you want the import to fall under, and note the number equivalent for this form.
    7. run it and wait. PMO, 1832 items, too about five minutes
    8. rejoice
