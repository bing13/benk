    Instructions To import Info Select items....

    Notes

    1. You can delete existing items (in proj 3) with: Item.objects.filter(project__id=3).delete()
    2. Then, be sure to create an anchor item for the project, parent=0, follower=0
    3. Import depends upon importISid's, which are cleared after each import
    4. Beware hogging too much CPU. Try using update instead of x.save()?

    Procedure

    1. export a chunk of an IS file to HTML
    2. move that to the same directory with parseIShtml.py. The main selector file should be in that directory, and the supporting files in a subdirectory of that directory.
    3. run parseIShtml.py over the tree of web pages
    4. use the resulting output file to enter here
    5. before you run the import, create the Project and its anchor item in Benk.
    6. run it and wait. 1832 items, took about five minutes
